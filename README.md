Módulo de Gestión de Personal (CRUD)
Este proyecto consiste en un módulo interactivo de administración de empleados desarrollado con tecnologías web estándar (Frontend). Permite gestionar los datos de personal en una interfaz dinámica y fluida.

Funcionalidades Principales 

Operaciones CRUD Completas:
- Create: Registro de nuevos empleados (Nombre, Puesto, Departamento).
- Read: Visualización de registros en una tabla dinámica en tiempo real.
- Update: Sistema de edición de registros existentes con carga de datos en el formulario.
- Delete: Eliminación selectiva de registros del listado.

Validación de Datos: 
- Lógica de control para asegurar que no se envíen campos vacíos, incluyendo retroalimentación visual al usuario mediante clases de error en CSS.

Interfaz Responsiva: 
- Diseño limpio y adaptable que facilita la gestión administrativa desde cualquier navegador.

Tecnologías Utilizadas

JavaScript (ES6+):
- Manipulación dinámica del DOM, gestión de eventos, lógica de estados mediante arreglos de objetos y funciones de filtrado.

HTML5:
- Estructura semántica para formularios y tablas de datos.

CSS3:
- Diseño personalizado con enfoque en la experiencia de usuario (UX) y validación visual.
